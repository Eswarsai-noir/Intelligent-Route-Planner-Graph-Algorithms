# Intelligent-Route-Planner-Graph-Algorithms
An Intelligent Route Planner built using Graph Data Structures, BFS, DFS, Dijkstra's Algorithm, and Priority Queue to find optimized routes between locations with optional graph visualization.



Overview

An Intelligent Route Planner built using **Graph Data Structures**, **BFS**, **DFS**, and **Dijkstra's Algorithm** to find the shortest path between locations. This project demonstrates how navigation and logistics systems optimize routes.

Features

* Graph Representation using Adjacency List
* BFS Traversal
* DFS Traversal
* Dijkstra's Shortest Path Algorithm
* Priority Queue (Min Heap)
* Route Summary Generation
* Graph Visualization (NetworkX & Matplotlib)

Tech Stack

* Python
* Heapq
* NetworkX
* Matplotlib


DSA Concepts Used

* Graphs
* Adjacency List
* BFS
* DFS
* Dijkstra Algorithm
* Priority Queue
